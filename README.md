# Moreno Cunha — Personal Portfolio (GitHub Pages)

This repository holds the static source files for **Moreno Cunha’s personal portfolio website**, built with plain HTML/CSS/JS and published via **GitHub Pages**.

---

## 🌐 Live Site

- Available at: `https://morenopc.com`
- Hosted at: `https://morenopc.github.io/`

---

## 📂 Repository Structure

- `index.html` — Main landing page  
- `static/` — CSS, JavaScript, fonts, and favicon assets  
- `media/` — Images used in the site (`picture/`, `project-cover/`, `project-image/`)  
- `CNAME` — Custom domain configuration (if applicable)

---

## ▶️ Run Locally

This is a static site, so you can just open `index.html` in a browser.  
For a local server (recommended for correct relative paths):

### ✅ Option A: Python (built-in)

```bash
cd morenopc.github.io
python3 -m http.server 8000
```
